                                Modos de operación
                            

                            

                                

                                    Chat
                                    Conversación directa con el modelo. Sin herramientas, solo texto.
                                

                                

                                    Agent
                                    El modelo puede ejecutar herramientas: leer archivos, correr comandos, editar código.
                                

                                

                                    Plan
                                    Genera un plan de pasos antes de actuar. Ideal para tareas largas.
                                

                            

                        


                        
                        

                            

                                
                                Skills
                            

                            
Los skills son instrucciones personalizadas que se inyectan al system prompt del agente. Se guardan como archivos .md en .agents/skills/ del workspace.
